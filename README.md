## 📌 Sobre o Projeto
Esta é uma **calculadora web** desenvolvida como parte dos meus estudos em desenvolvimento front-end.  
O projeto permite realizar operações matemáticas básicas com uma interface simples, intuitiva e responsiva.

## ✨ Funcionalidades
- ➕ Soma  
- ➖ Subtração  
- ✖️ Multiplicação  
- ➗ Divisão  
- 🧹 Botão de limpar (reset)  
- 📱 Layout responsivo  

## 🛠️ Tecnologias Utilizadas
- HTML5  
- CSS3  
- JavaScript  

## 🎯 Objetivo
Este projeto foi desenvolvido com o objetivo de:
- Praticar lógica de programação  
- Trabalhar manipulação do DOM  
- Utilizar eventos com JavaScript  
- Melhorar habilidades em front-end
