<h1> < Calculadora Simples /> </h1>
<p>Este projeto é uma calculadora desenvolvida como prática de HTML, CSS e JavaScript. O objetivo foi reforçar a lógica de programação e a manipulação de eventos, além de criar uma interface moderna e responsiva para mobile. Durante o desenvolvimento, trabalhei o
  rganização de código, estilização e usabilidade, transformando conceitos básicos em uma aplicação funcional.</p>
